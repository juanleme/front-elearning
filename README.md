# FRONT-END - Sistema Gamification e-learning#


**Dependências do projeto**

* NodeJS


**Instalação**

Na pasta raiz do projeto é necessário aplicar os seguintes 


```

npm update -g

npm cache clean

npm install -g gulp

npm install


```

Para executar o servidor use o seguinte comando

```

gulp server

```
